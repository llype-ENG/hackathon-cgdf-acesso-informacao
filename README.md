# hackathon-cgdf-acesso-informacao
Repositório destinado ao desenvolvimento da solução para o desafio do Hackathon da Controladoria-Geral do Distrito Federal (CGDF), no tema Acesso à Informação, com foco na identificação e filtragem de dados sensíveis para classificar propostas como públicas ou privadas, alinhando transparência e proteção de dados.
